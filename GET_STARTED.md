# Get Started in 3 Steps

## Step 1: Install Extension (1 minute)

1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer mode** (toggle top-right)
3. Click **Load unpacked**
4. Select the `jira-timesheet-dashboard` folder
5. Done! ‚úÖ

---

## Step 2: Use It (30 seconds)

1. Open your Jira Cloud (e.g., `yourcompany.atlassian.net`)
2. Click the extension icon in Chrome toolbar
3. Select date range (Today, This Week, This Month, or Custom)
4. Click **Load**
5. View your worklogs with per-day breakdown! üéâ
6. Click any issue to open it in Jira
